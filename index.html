O que é HTML?

HTML (HyperText Markup Language) é a linguagem de marcação padrão usada para criar páginas web. Ele define a estrutura e o conteúdo de um documento da web usando elementos e tags.

Estrutura Básica de uma Página HTML

Uma página HTML é composta por elementos básicos organizados de forma hierárquica. A estrutura básica inclui:

  • <!DOCTYPE html>: Define o tipo do documento.
  • <html>: Elemento raiz que contém todo o conteúdo.
  • <head>: Contém metadados, como título e links para arquivos externos.
  • <body>: Contém o conteúdo visível da página.

Elementos Principais

Os principais elementos do HTML incluem:

  • <p>: Parágrafos.
  • <a>: Links.
  • <h1> a <h6>: Cabeçalhos.
  • <img>: Imagens.
  • <ul> e <ol>: Listas não ordenadas e ordenadas.
